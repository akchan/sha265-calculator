SHA-256 Calculator

About

This app calculates a hash of given text using the SHA-256 algorithm. You can always get same hash from same text. But it is very difficult to estimate original text from hash.

Hash calculation depends on jsSHA which is a JavaScript implementation of the SHA family. So, ensure that JavaScript is enabled in your web browser.

Each line in the 'TEXT' field corresponds to that of the 'Hash (HEX-encoded)' field.
